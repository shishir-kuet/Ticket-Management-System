================================================================================
                            TECHNICAL DETAILS
================================================================================

RESPONSE FORMAT:
All responses are JSON objects with:
- success: boolean
- response: string (markdown formatted)
- quick_actions: array of action objects
- message: string (for errors)

QUICK ACTION FORMAT:
{
  "text": "Display Text",
  "action": "action_name"
}

MESSAGE PROCESSING:
1. Check for quick actions first (fastest response)
2. Try AI service if available (OpenAI integration)
3. Fall back to rule-based keyword matching
4. Return default response if no matches

KEYWORD MATCHING:
- Case insensitive
- Uses containsWords() function
- Checks for any keyword presence in user message
