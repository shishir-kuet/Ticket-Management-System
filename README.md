Ticket Management System

A simple and efficient Ticket Management System built with PHP and MySQL to organize, track, and resolve support tickets. Designed for small teams and helpdesk workflows.

🚀 Highlights

Create, assign, and track support tickets

User roles: Admin, Staff, Customer

Status updates (Open, Pending, Resolved)

Clean PHP + MySQL implementation

Ready for real deployment or further customization

🧠 Built With

PHP — Core backend logic

MySQL — Database storage

HTML/CSS — Frontend interface

Minimal dependencies for easy setup

📦 Quick Setup

Clone the repo

git clone https://github.com/shishir-kuet/Ticket-Management-System.git
cd Ticket-Management-System

Database

Create a MySQL database

Import provided SQL (if included) or run schema setup

Configure

Update database credentials in config/auth files

Launch

Place in web server directory (e.g., XAMPP/WAMP htdocs)

Open in browser: http://localhost/Ticket-Management-System

🛠 Features Overview
Feature	Description
📝 Ticket Creation	Users can submit and describe issues
👥 User Roles	Different access for admins, staff, customers
🔁 Ticket Status	Track progress from open to resolved
📊 Dashboard	Summary of active and past tickets
🧩 How It Works

Users log in and submit tickets. Admin/staff can assign, update, and close tickets. The database stores all interactions for easy tracking.
